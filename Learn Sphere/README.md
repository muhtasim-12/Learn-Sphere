# KnowLink

A simple prototype for a universal knowledge graph website.

## What it does

- Search books, topics, authors, sources and concepts
- Shows related knowledge nodes
- Shows legal availability sources
- Creates a basic learning roadmap

## How to run

Open index.html in any browser.

## Future upgrades

- Add backend database
- Connect Google Books API
- Connect Open Library API
- Add user accounts
- Add used book marketplace
- Add seller/lender listings
- Add AI learning roadmap generator
